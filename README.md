# Sukanya Interactive Worlds Site

A Vite + React project built as a two-world, mobile-first personal site.

## Run locally
```bash
npm install
npm run dev
```

## What is already done
- Multi-file structure
- Mobile + desktop responsive layouts
- Animation-heavy landing + two world flows
- Five custom illustrated portrait cards inspired by the uploaded photos
- Sharpened copy and more personal romantic / friendship arcs

## What you may still want to change later
- Add your real phone number in `src/data/siteData.js`
- Replace playlist titles with actual songs or Spotify links
- Adjust any copy line you want to make more personal
